Jekyll

  • https://jekyllrb.com/
  • Written in Ruby
  • Favors conventional directories like _posts
  • Very focused on blogs

Hexo

  • https://hexo.io
  • Written in JavaScript
  • Migrate from WordPress, Joomla, RSS, etc.
  • Use a build command via hexo new

Hugo

  • http://gohugo.io/
  • Written in Go
  • Fast
  • Did I mention it's fast!?
  • More freeform than blog focused
